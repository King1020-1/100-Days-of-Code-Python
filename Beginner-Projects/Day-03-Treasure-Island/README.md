# Day 3 – Treasure Island
This is my Day 3 Project

---
## Description
This is an story where the user needs to solve different puzzles, make logical choices,
correct choices leads to winning but incorrect choices will make you lose

---
## Learnings
1. Using ASCII art to make the program look better
2. Using `if` and `else` and nested if else
3. Using `while` loop to let user replay the game
4. Using `""" """` to make a string across multiple lines
5. Using `r" "` to treat \ as a normal character rather than escape sequence
